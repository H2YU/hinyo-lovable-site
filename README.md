# HINYO Lovable Site

Premium B2B Pilates equipment brand site for HINYO.

## Brand

- Brand: HINYO
- Positioning: Movement Space Brand
- Tagline: Create Better Movement Spaces
- Core product: HINYO Spiral Tower System

## Pages / Sections

This project is a responsive single-page React site with sections that map to the requested website pages:

- Home
- Spiral Tower
- Equipment
- Studio Setup
- About
- Contact

## Local Preview

```bash
npm install
npm run dev
```

## Lovable Import

Import this repository into Lovable as a React/Vite project. Placeholder media blocks are intentionally used so real HINYO product images and videos can be replaced later.
